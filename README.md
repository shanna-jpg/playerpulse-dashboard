# PlayerPulse Analytics Dashboard

This project is a sports analytics dashboard built using Python, Streamlit, and SQLite.

## Features
- Displays team, player, game, and performance data
- Uses a relational database to organize statistics
- Interactive dashboard interface built with Streamlit

## Tech Stack
- Python
- Streamlit
- SQLite
- Pandas

## How to Run
pip install streamlit pandas
streamlit run playerpulse_app.py.py
